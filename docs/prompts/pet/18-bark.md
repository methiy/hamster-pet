# 18 · 帧动画 · bark（吠叫）

> **对应文件**：`src/assets/sprites/bark-0.png` … `bark-3.png`
> **帧数**：4
> **单帧尺寸**：128 × 128 PNG RGBA，透明背景
> **循环**：是
> **帧率**：8 fps

## 触发场景（代码侧）

- 召唤 pet（Ctrl+Shift+P）到达时短暂播放 bark 后回 idle
- 间隔提醒触发时（和跑过去的动作配合）
- 某些 reaction 时替代 happy

## 动作描述

柯基**正面**，**抬头张嘴吠叫**两声（4 帧 = 两个"汪"）。

- 第 0 帧：嘴闭合（准备吸气），头部略低
- 第 1 帧：嘴**大张**（第一声"汪"），头**略上抬**，胸腔鼓起
- 第 2 帧：嘴闭合（声音之间的间隔），头回到中间位置
- 第 3 帧：嘴**再次大张**（第二声"汪"），头再上抬

核心特征：
- **两次明显的张嘴**（第 1、3 帧），同一动作节奏重复
- **嘴张开时能看到牙齿和舌头**（粉色舌根 + 简单的白色上下各 2-3 颗小牙）
- **音符符号**："♪" 或 "woof" 文字样化的像素符号飘在头旁（可选）
- 身体姿态基本不动，全部是头部和嘴部的变化
- 尾巴可以直立翘起，随每次"汪"而微摆

## 注意

- 循环动画：两次吠叫为一个完整循环，衔接自然。
- 嘴型要可爱，不要画成凶恶咧牙。
- 音符符号只占头部右上方很小的区域，不要抢戏。
