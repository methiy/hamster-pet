# 12 · 帧动画 · landed（屁股蹲掉地）

> **对应文件**：`src/assets/sprites/landed-0.png` … `landed-3.png`
> **帧数**：4
> **单帧尺寸**：128 × 128 PNG RGBA，透明背景
> **循环**：**否**（一次性播放，结束后切到 idle）
> **帧率**：8 fps（0.5 秒播完）

## 触发场景（代码侧）

`App.vue` 的 `onGrabEnd`：用户松开拖拽的 pet，窗口会下掉 + 回弹。新 sprite 用来表达 pet 落地时**屁股先着地、被震懵**的那一瞬间。

## 动作描述

柯基**正面**，落地时**屁股先着地**、四肢摊开，然后身体震一下爬起来。

- 第 0 帧：落地的瞬间，**屁股重砸地面**：后腿压扁张开（青蛙坐姿），前腿支地，身体上半身前倾
- 第 1 帧：身体被震动**往上微弹**（整体上抬 2-3 像素），头部更突出（夸张）
- 第 2 帧：身体回落到地面，但仍是坐在屁股上的姿势；前腿收回
- 第 3 帧：身体开始站起（接近 idle 的基础姿态，屁股离地但还没完全站直）

核心特征：
- **第 0 帧最有冲击力**：屁股扁、四腿张、眼睛瞪圆、嘴大张（像 "哎哟！"）
- 可以有"星星/晕眩"圆圈绕头（可选，2-3 像素小点）
- **第 3 帧**要视觉上能**平滑过渡**到 `idle-0`（类似姿态）

## 注意

- 非循环，**第 3 帧结束后**代码切回 idle，所以第 3 帧姿态应当**接近 idle-0**。
- 屁股坐地时要表现出"被砸懵"的幽默感，不要画得很痛苦。
