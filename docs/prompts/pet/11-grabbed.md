# 11 · 帧动画 · grabbed（被拎起来）

> **对应文件**：`src/assets/sprites/grabbed-0.png` … `grabbed-3.png`
> **帧数**：4
> **单帧尺寸**：128 × 128 PNG RGBA，透明背景
> **循环**：是
> **帧率**：5 fps

## 触发场景（代码侧）

`App.vue` 的 `onGrabStart`：用户长按 pet 窗口拖拽它。当前只用 CSS 拉长做视觉效果；新 sprite 用来表达"被拎起来悬空挣扎"的样子。

## 动作描述

柯基**正面**，被**隐形的手从后颈拎起悬空**，四腿垂下微微晃动。身体**略微拉长**（从肩部开始纵向被拉高 5-8 像素）。

- 第 0 帧：四腿垂直下垂，正常位置
- 第 1 帧：四腿略向左摆（约 3 像素）
- 第 2 帧：四腿回到中间
- 第 3 帧：四腿略向右摆，准备回到第 0 帧

核心特征：
- **肩部以上**被向上拎起的感觉：头部/脖子与身体之间有一点拉伸
- **四腿松弛垂下**，像小狗被铲屎官拎起来时脚悬空不动
- **嘴微张**，可以画成 "><" 或无奈的表情
- **眼睛半眯/无奈**（不开心但也没挣扎）
- **尾巴下垂**（不翘），像跟着身体垂下

## 注意

- 循环动画：第 0/2 帧四腿居中，第 1/3 帧左右摆，衔接就是一个钟摆循环。
- 整个身体的朝向是**正面略微俯视** —— 因为是从"头顶拎起"的视角。
- 注意画布安全区：四腿不要超出画布下边缘。
