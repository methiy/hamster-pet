# 08 · 帧动画 · idle（待机呼吸）

> **对应文件**：`src/assets/sprites/idle-0.png` … `idle-3.png`
> **帧数**：4
> **单帧尺寸**：128 × 128 PNG RGBA，透明背景
> **循环**：是（第 3 帧要能无缝回到第 0 帧）
> **帧率**：4 fps（播放一轮 1 秒）

---

## 动作描述

柯基**站立**（或坐姿均可，固定同一姿态），**轻微呼吸起伏**。身体不移动、不换朝向，整个动画只表达"活着、在呼吸"。

- 第 0 帧：身体最低点（胸腔收回）
- 第 1 帧：向上 1-2 像素
- 第 2 帧：身体最高点（胸腔鼓起）
- 第 3 帧：向下 1-2 像素，回到接近第 0 帧的姿态

全程**眼睛睁开**，嘴闭合或微张。尾巴可以有 1-2 像素的轻微摆动（可选，不是硬要求）。

## 注意

- 整组帧视觉差异**非常小**，靠垂直 1-2 像素的位移和胸腔轮廓的轻微变化。帧与帧之间不要出现突兀的动作。
- 第 3 帧到第 0 帧的回卷必须平滑。
