# 装饰品图标提示词

统一要求：可爱像素风小图标，64×64 像素，透明背景 PNG，物品居中，轮廓清晰，柔和的阴影，单个物品。

---

## 1. crown（小皇冠）

一顶精致小巧的金色皇冠，适合小仓鼠戴的迷你尺寸。皇冠顶部有三个小尖角，每个尖角顶端镶着一颗小红宝石，金属表面闪着金色高光。可爱像素风图标，64×64像素，透明背景。

## 2. sunglasses（墨镜）

一副酷酷的小墨镜，仓鼠尺寸。黑色镜框圆圆的，镜片上有一点淡蓝色的反光效果，显得很时髦。可爱像素风图标，64×64像素，透明背景。

## 3. bow（蝴蝶结）

一个可爱的粉色缎带蝴蝶结，蓬蓬松松的，中间有一个结扣。柔柔的粉红色，边缘有稍浅的高光。可爱像素风图标，64×64像素，透明背景。

## 4. bell（小铃铛）

一个金灿灿的小铃铛，顶部系着一个红色蝴蝶结做的挂绳。铃铛表面圆润发亮，底部有一条开口缝隙，能看到里面的小珠子。可爱像素风图标，64×64像素，透明背景。

## 5. backpack（小背包）

一个迷你的绿色探险小书包，鼓鼓囊囊装得满满的，有一个小扣子和可见的肩带。森林绿色为主，棕色皮革装饰扣带。可爱像素风图标，64×64像素，透明背景。

## 6. wreath（花环）

一个用各种小花编织而成的花环头饰，粉色、黄色、紫色的迷你小花和绿叶交织在一起，清新又春意盎然。可爱像素风图标，64×64像素，透明背景。
