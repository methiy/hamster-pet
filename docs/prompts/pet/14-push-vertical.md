# 14 · 帧动画 · push-vertical（竖直方向推拉窗口）

> **对应文件**：`src/assets/sprites/push-v-0.png` … `push-v-3.png`
> **帧数**：4
> **单帧尺寸**：128 × 128 PNG RGBA，透明背景
> **循环**：是
> **帧率**：6 fps

## 触发场景（代码侧）

`usePushAnimation.ts` 里 push 方向 `up`/`down` 的情况。**上下方向共用这组 sprite**，代码在 `down` 时用 CSS 做**垂直翻转**。所以本文件画**向上推**即可（柯基双爪举向画面顶部）。

## 动作描述

柯基**正面**或**略俯视角**，后腿站稳、**双前爪向上举**用力顶住头顶上方一个看不见的平面（要推的窗口）。

- 第 0 帧：双前爪向上举起（举到头顶偏上位置），身体半蹲蓄力
- 第 1 帧：发力瞬间（双前爪更伸展，身体直立、踮起后腿）
- 第 2 帧：身体略收回（发力回荡），但前爪仍抵在头顶方向
- 第 3 帧：回到接近第 0 帧的蓄力状态

核心特征：
- **双前爪高举过头顶**，指向画布上边缘（爪子要贴到画布接近上边 8-12 像素的位置）
- **身体正面朝向观众**（和 idle 同视角）
- **后腿向下踮起**用力支撑
- **脸部朝上看**：头略后仰，眼睛向上看
- **嘴咬紧或张开用力**：表情紧绷
- 尾巴**朝下绷紧**（配重）

## 注意

- 循环动画，蓄力→推→回弹→蓄力。
- 代码里 dir='down' 时 CSS 会垂直翻转（`scaleY(-1)`）显示这组 sprite，不用另外画。
- 画布安全区：前爪**不要**超出上边缘；给它至少 8 px padding。
