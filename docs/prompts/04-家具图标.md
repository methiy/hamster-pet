# 家具图标提示词

统一要求：可爱像素风小图标，64×64 像素，透明背景 PNG，物品居中，轮廓清晰，柔和的阴影，单个物品。

---

## 1. wheel（仓鼠跑轮）

一个彩色的仓鼠跑轮，橙黄色的圆形转轮架在蓝色金属支架上，稍微倾斜一个角度能看到跑步的内侧面。造型圆润可爱。可爱像素风图标，64×64像素，透明背景。

## 2. nest（温暖小窝）

一座温馨的仓鼠小木屋，暖棕色的木头材质，有一个圆拱形的小门洞，屋顶有一根小烟囱，门洞里能看到里面铺着柔软的棉花垫子。小屋造型像一个迷你乡村小屋，特别温暖可爱。可爱像素风图标，64×64像素，透明背景。

## 3. swing（小秋千）

一架迷你的绳索秋千，木板做的座位用两根麻绳吊在一个小木框架上。天然的木头和麻绳颜色，造型简单又有趣。可爱像素风图标，64×64像素，透明背景。

## 4. sunflower_pot（向日葵盆栽）

一个小陶土花盆里种着一朵灿烂的向日葵。棕红色的陶土盆，绿色的茎和叶子，明亮的黄色花瓣围着棕色的花心，充满阳光的感觉。可爱像素风图标，64×64像素，透明背景。

## 5. treasure_chest（宝藏箱）

一个小木头宝箱微微打开着，从缝隙里可以看到金币和彩色宝石在里面闪闪发光，金色的金属铰链和锁扣。深棕色的木头配金色装饰，箱子周围有一两点魔法闪光特效。可爱像素风图标，64×64像素，透明背景。
