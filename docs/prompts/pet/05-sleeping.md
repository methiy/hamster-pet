# 12 · 帧动画 · sleeping（睡觉）

> **对应文件**：`src/assets/sprites/sleeping-0.png` … `sleeping-3.png`
> **帧数**：4
> **单帧尺寸**：128 × 128 PNG RGBA，透明背景
> **循环**：是（第 3 帧要能无缝回到第 0 帧）
> **帧率**：3 fps（播放一轮约 1.33 秒，故意慢）

---

## 动作描述

柯基**侧躺**或**蜷卧**，**闭眼**，呼吸起伏非常缓慢。**视角侧面或侧面略俯**（整只狗卧下的轮廓最好看）。

- 第 0 帧：身体最收（胸腔呼出状态）
- 第 1 帧：身体略起（吸气中）
- 第 2 帧：身体最舒展（胸腔吸气最满）
- 第 3 帧：身体略收（呼气中），回到接近第 0 帧

**核心特征**：
- **眼睛全程紧闭**（一条线 `-` 表示）
- 嘴闭合，**可以有一个小 "z z" 或 "💤" 像素化字符飘在头顶**（可选，放在每帧右上方的透明区域内）
- 尾巴**贴身**、不动
- 耳朵**下垂或贴头**（睡觉状态）

## 注意

- 帧之间差异小（睡觉的呼吸幅度只有 1-2 像素）。
- 身体轮廓**不要分腿分脚画太清晰**——蜷卧时就是一坨毛绒绒的轮廓。
- 如果画 "💤"，四帧位置可以略有漂移（z 字飘上去的感觉），但不要太大。
