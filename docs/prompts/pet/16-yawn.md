# 16 · 帧动画 · yawn（打哈欠）

> **对应文件**：`src/assets/sprites/yawn-0.png` … `yawn-3.png`
> **帧数**：4
> **单帧尺寸**：128 × 128 PNG RGBA，透明背景
> **循环**：**否**（一次性播放，结束后切回 idle 或 sleeping）
> **帧率**：5 fps（0.8 秒播完）

## 触发场景（代码侧）

idle 状态维持一段时间后随机插入 yawn → 再回到 idle 或过渡到 sleeping。用于让 pet 看起来更有生命感，不是一直保持同一个呼吸动作。

## 动作描述

柯基**正面**，打一个大哈欠。嘴张大 → 最大 → 闭合。

- 第 0 帧：嘴闭合（idle 基准姿态），眼睛开始半眯
- 第 1 帧：嘴开始张开（半张），眼睛继续眯
- 第 2 帧：**嘴张到最大**（最核心的哈欠帧），舌头**卷起向外吐出**一小截，眼睛**闭合或眯成弯月**
- 第 3 帧：嘴闭合回基准，眼睛半睁（准备回到 idle 或过渡睡觉）

核心特征：
- **第 2 帧是标志性的大哈欠帧**：嘴占脸部高度的 1/2 左右，舌头粉色卷曲
- **眼睛随哈欠同步闭合**（第 2 帧眼缝最窄）
- **一只前爪可以抬起来**像人类打哈欠时遮嘴的动作（可选，不是必须）
- 身体几乎不动（就是头部特写）

## 注意

- 非循环：第 3 帧结束即完结。
- **第 3 帧的姿态要接近 idle-0**，便于切回去。
- 嘴张得越大越搞笑，但不要离谱到把整个脸占满。
