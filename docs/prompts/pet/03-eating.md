# 10 · 帧动画 · eating（吃东西）

> **对应文件**：`src/assets/sprites/eating-0.png` … `eating-5.png`
> **帧数**：6
> **单帧尺寸**：128 × 128 PNG RGBA，透明背景
> **循环**：是（第 5 帧要能无缝回到第 0 帧）
> **帧率**：8 fps（播放一轮 0.75 秒）

---

## 动作描述

柯基**坐姿**或**趴姿**，正面视角，**低头咀嚼**。动作聚焦在**嘴部开合 + 头部轻微上下**，身体其他部分几乎不动。

- 第 0 帧：嘴闭合，头部平放
- 第 1 帧：嘴微张，头略低
- 第 2 帧：嘴大张（咬合最开的瞬间）
- 第 3 帧：嘴微张（咬合回收中）
- 第 4 帧：嘴闭合，头略抬
- 第 5 帧：嘴闭合，头回到初始位置（平滑回到第 0 帧）

**嘴部是动画核心**，应有明显像素级变化：闭→微张→大张→微张→闭。脸颊**可以在第 2-3 帧鼓起一点**（嚼东西的感觉）。

**不要画食物**：画面里只有柯基本身，食物由另一层渲染，不能内嵌进精灵。

## 注意

- 耳朵**可以随头部上下轻微摆**（1 像素位移），不是必须。
- 眼睛可以是半眯（"享受"的表情），也可以睁开，保持同一帧内两只眼睛对称。
