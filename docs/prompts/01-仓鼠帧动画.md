# 仓鼠帧动画提示词

统一要求：可爱 Q 版像素风仓鼠，透明背景 PNG。仓鼠主色橙棕色，浅色肚皮，黑色圆豆眼，粉色小鼻子，圆滚滚的身体，短小的四肢和尾巴。

**全部 8 个状态放在一张大图里**，每帧 128×128 像素，排列方式如下：

```
第1行：idle（4帧）     → 128×4 = 512px 宽
第2行：eating（6帧）   → 128×6 = 768px 宽
第3行：sleeping（4帧） → 128×4 = 512px 宽
第4行：running（4帧）  → 128×4 = 512px 宽
第5行：happy（6帧）    → 128×6 = 768px 宽
第6行：hiding（4帧）   → 128×4 = 512px 宽
第7行：adventure_out（6帧）  → 128×6 = 768px 宽
第8行：adventure_back（6帧） → 128×6 = 768px 宽

整张图尺寸：768×1024（宽取最大值768，高 128×8=1024）
帧数不足6帧的行，右侧留空即可。
```

---

## 每行内容描述

### 第1行：idle（待机）4帧，循环播放

一只可爱的Q版像素风小仓鼠端坐着，面朝右前方，圆滚滚的身体，小爪子放在肚皮上，黑豆眼睛亮晶晶的。第一帧正常坐姿；第二帧身体微微鼓起像在吸气；第三帧恢复正常；第四帧小耳朵轻轻抖动一下。整体感觉是安静乖巧地坐着，偶尔动一下。

### 第2行：eating（吃东西）6帧，循环播放

一只可爱的Q版像素风小仓鼠在吃东西，双爪捧着一颗小瓜子送到嘴边，腮帮子越吃越鼓。第一帧捧着食物靠近嘴巴；第二帧张嘴咬住；第三帧嚼嚼嚼腮帮子微微鼓起；第四帧继续嚼；第五帧腮帮子明显鼓起来了；第六帧吞下去腮帮子稍微消一点。表情满足幸福。

### 第3行：sleeping（睡觉）4帧，循环播放

一只可爱的Q版像素风小仓鼠蜷成一团在睡觉，眼睛闭着弯成两道月牙，小爪子缩在身体里，尾巴卷着环绕身体。第一帧安静睡着头顶飘着一个小z；第二帧身体微微起伏像在呼吸z往上飘；第三帧身体落下出现一个大Z；第四帧大Z渐隐新的小z又冒出来。整体感觉温馨安详。

### 第4行：running（跑步）4帧，循环播放

一只可爱的Q版像素风小仓鼠在原地快速奔跑，侧面视角，小短腿交替蹬地，耳朵随着跑动上下晃，表情开心兴奋。经典跑步循环：左腿前右腿后身体上弹、腿交叉身体下沉、右腿前左腿后身体上弹、腿交叉身体下沉。身后带一两条速度线。

### 第5行：happy（开心）6帧，播放一次

一只可爱的Q版像素风小仓鼠开心得跳起来了！眼睛变成星星眼，嘴巴大笑，小爪子举高高。第一帧蹲下蓄力；第二帧蹬地起跳双手展开；第三帧跳到最高点周围冒出星星闪光特效最灿烂的笑容；第四帧开始下落；第五帧落地轻微弹跳；第六帧站定开心摇尾巴还有残余的小星星。

### 第6行：hiding（害羞）4帧，播放一次

一只可爱的Q版像素风小仓鼠被吓到然后害羞地捂脸。第一帧受惊表情耳朵竖起来眼睛睁大；第二帧急忙用两只小爪子捂住脸；第三帧从爪子缝里偷偷露出一只眼睛偷看耳朵耷拉下来耳尖微微泛红；第四帧完全捂住脸不看了耳朵上有害羞的红晕。

### 第7行：adventure_out（出发冒险）6帧，播放一次

一只可爱的Q版像素风小仓鼠背着小绿色书包，坚定地向右走出画面去冒险。第一帧面向观众挥挥小爪子说再见；第二帧转身面朝右边；第三帧迈步向右走背上的小书包清晰可见表情坚毅；第四帧继续走走得更远了；第五帧大部分身体已经走出画面右边缘；第六帧只剩一截小尾巴尖或者完全走出去了。

### 第8行：adventure_back（冒险归来）6帧，播放一次

一只可爱的Q版像素风小仓鼠从左边兴冲冲地跑回来，背着鼓鼓囊囊装满东西的小书包。第一帧只有一只小爪子和耳尖从画面左边冒出来；第二帧小仓鼠开心地跑进画面笑容灿烂；第三帧继续跑书包随着跑动颠来颠去；第四帧急刹车滑行到画面中间；第五帧站稳了举起一只小爪子炫耀找到的宝贝；第六帧得意的胜利姿势书包放在脚边周围冒出小星星。
