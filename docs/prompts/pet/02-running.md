# 09 · 帧动画 · running（快步跑）

> **对应文件**：`src/assets/sprites/running-0.png` … `running-3.png`
> **帧数**：4
> **单帧尺寸**：128 × 128 PNG RGBA，透明背景
> **循环**：是（第 3 帧要能无缝回到第 0 帧）
> **帧率**：6 fps（播放一轮约 0.66 秒）

---

## 动作描述

柯基**向右侧方向快步小跑**，四腿交替（柯基腿很短，跑起来显得很萌的"碎步")。整个动画就是一个完整的"一步"循环。

**视角：侧面（朝右），不是正面**——这是跑步动画的必要视角；全身朝向一致，尾巴在身体后方。

- 第 0 帧：前腿左腿抬起、后腿右腿抬起（对角支撑期）
- 第 1 帧：四脚离地（腾空最高点），身体略前倾
- 第 2 帧：前腿右腿抬起、后腿左腿抬起（另一组对角）
- 第 3 帧：四脚离地（另一次腾空），为下一循环做准备

身体**可以有 1-2 像素的上下颠簸**（跑动时自然的 bob），耳朵**可以向后飘一点**。尾巴**向上翘**，可随步频轻微摆动。

## 注意

- 跑步是循环动画，**帧与帧动作必须对称且连续**；如果你看到第 0 帧和第 2 帧是同一姿势，说明循环没做对。
- 侧面视角的柯基，双侧对称身体只看到一侧耳朵、一侧眼睛、两条近腿 + 两条远腿。远侧腿稍暗（像素艺术可用深一度颜色表现深度）。
