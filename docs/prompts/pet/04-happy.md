# 11 · 帧动画 · happy（开心跳跃）

> **对应文件**：`src/assets/sprites/happy-0.png` … `happy-5.png`
> **帧数**：6
> **单帧尺寸**：128 × 128 PNG RGBA，透明背景
> **循环**：是（第 5 帧要能无缝回到第 0 帧）
> **帧率**：8 fps（播放一轮 0.75 秒）

---

## 动作描述

柯基**原地小跳两次**，表达兴奋。正面视角，身体不换朝向。

- 第 0 帧：站立基准（四脚着地）
- 第 1 帧：蓄力（前腿微蹲，重心下沉 2-3 像素）
- 第 2 帧：跳跃最高点（整体上抬 6-10 像素，四脚离地）
- 第 3 帧：落地瞬间（回到接近第 0 帧的位置，身体略震）
- 第 4 帧：再次蓄力（浅一点）
- 第 5 帧：再次跳起（浅一点，抬起约 3-5 像素），为回到第 0 帧做准备

**核心特征**：
- **嘴大张**（像在笑）贯穿全程
- **舌头吐出**一小截，粉色
- **眼睛眯成 ^_^** 或星星眼
- **尾巴剧烈摇摆**（每帧朝不同方向）
- 第 2 帧最高点时，**耳朵向上翘起**

## 注意

- 这是"开心"动画，情绪浓度要高 —— 嘴笑开、眼睛弯、尾巴甩。
- 跳高度不要太夸张，整个角色始终在画布内（四周 8px 安全区别越过）。
