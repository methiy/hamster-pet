# 15 · 帧动画 · angry（生气）

> **对应文件**：`src/assets/sprites/angry-0.png` … `angry-3.png`
> **帧数**：4
> **单帧尺寸**：128 × 128 PNG RGBA，透明背景
> **循环**：是
> **帧率**：6 fps

## 触发场景（代码侧）

`useActivityReaction.ts` 检测到用户沉迷某项活动（长时间 coding/video/gaming 不理 pet），可以替代现有的 reactionState 部分，表达"抗议"。也可以在间隔提醒触发时使用。

## 动作描述

柯基**正面**，气鼓鼓的表情。身体微微颤抖、**脸颊鼓起**，像生闷气。

- 第 0 帧：身体正常位置，**眉毛压低**（倒八字），嘴扁平
- 第 1 帧：身体轻微左晃（1-2 像素），**脸颊鼓起**（腮部 2-3 像素外凸）
- 第 2 帧：回到中间，腮仍鼓
- 第 3 帧：身体轻微右晃，准备回到第 0 帧

核心特征：
- **倒八字眉**：最核心的"生气"特征，贯穿四帧
- **脸颊鼓起**：第 1-3 帧保持气鼓鼓
- **蒸汽符号**：头两侧可以有 1-2 个像素化的 "💢" 或漫画气愤符号（黑色小斜线聚在一点），飘动位置随帧变化
- **眼睛圆睁或眯起怒视**
- 尾巴**竖起毛炸**（蓬松有刺的感觉）

## 注意

- 循环动画，左右晃幅度要小（就是"气得浑身发抖"的细微颤动）。
- 表情必须**可爱生气**，不要凶恶。目标是让人觉得"哈哈它在生气真可爱"。
