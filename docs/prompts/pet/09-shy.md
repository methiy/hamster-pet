# 09 · 帧动画 · shy（害羞）

> **对应文件**：`src/assets/sprites/shy-0.png` … `shy-3.png`
> **帧数**：4
> **单帧尺寸**：128 × 128 PNG RGBA，透明背景
> **循环**：是
> **帧率**：5 fps

## 触发场景（代码侧）

`triggerReaction('shy', ~2000)`：被夸奖 / 被轻拍 / 主人点击头部区域时偶发；
也可以偶尔替代 idle 作为随机 idle 变体。

## 动作描述

柯基**正面**站立，脸部扭向一边、**脸颊发红**，像在害羞。

- 第 0 帧：正脸，表情刚开始变化（嘴微抿）
- 第 1 帧：头**略向左侧偏** 3-5 像素，腮红**开始显色**（两块浅粉色 2×2 像素块出现在脸颊）
- 第 2 帧：头继续偏，腮红**最深**（更明显的粉红色），眼睛半眯向下
- 第 3 帧：头回到接近正面，但腮红仍在，准备回到第 0 帧

核心特征：
- **腮红**是这个动作的视觉核心，必须清晰可见
- **一只前爪可以抬起来遮嘴**（可选，增加"害羞捂嘴"感）
- 尾巴微微摇（幅度比 idle 大一点）

## 注意

- 这是**循环**动画：第 3 帧到第 0 帧之间要自然衔接。
- 腮红第 0 帧为空、3 帧中期最浓、第 3 帧残留——让人一眼看出"正在脸红"的过程。
