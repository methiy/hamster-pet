# 13 · 帧动画 · hiding（躲藏）

> **对应文件**：`src/assets/sprites/hiding-0.png` … `hiding-3.png`
> **帧数**：4
> **单帧尺寸**：128 × 128 PNG RGBA，透明背景
> **循环**：**否**（一次性播放，停在第 3 帧）
> **帧率**：6 fps（播放一遍约 0.66 秒）

---

## 动作描述

柯基从**正常站立**的姿态**缩起来躲藏**，像受到惊吓、身体压低、耳朵收起。正面视角，身体不换朝向。

- 第 0 帧：正常站姿（同 idle 基准）
- 第 1 帧：开始下蹲，头部略低，耳朵开始向后倒
- 第 2 帧：身体明显缩低（高度减少约 1/3），耳朵完全贴在头上，眼睛半眯
- 第 3 帧：完全缩成一团、耳朵贴头、眼睛瞪大或紧闭，尾巴夹在身下

**核心特征**：
- 每一帧身体整体**高度逐步变低**（向下压缩）
- 耳朵**从立起→贴头**渐进
- 第 3 帧是"最躲藏"的终态（动画停在这一帧直到其他状态接管）
- 尾巴第 0 帧自然下垂，第 2-3 帧**夹在后腿之间**

## 注意

- 这是非循环动画 —— 第 3 帧后不会自动回到第 0 帧，所以第 3 帧本身要能作为**静态海报**看（够明显、表达完整的"缩起来"）。
- 头部别因为压低而移出画布下边缘；保留 8 px padding。
